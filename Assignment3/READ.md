## Brief Description
This assignment bridges the backend blockchain logic with a user-facing frontend. It features a web interface where users can connect their MetaMask wallet to read from and write transactions to a deployed smart contract directly from their browser.

## Tech Stack Used
* **Frontend:** HTML, CSS, JavaScript
* **Web3 Library:** Ethers.js (or Web3.js)
* **Wallet:** MetaMask

## How to Run
1. Ensure your smart contract is already deployed on a test network.
2. Open the main JavaScript file (`app.js` or similar) in this folder.
3. Replace the placeholder variables for `contractAddress` and `contractABI` with your actual deployed contract details.
4. Open the `index.html` file using a local web server (like the "Live Server" extension in VS Code).
5. Open the site in a browser with the MetaMask extension installed, click "Connect Wallet", and interact with the UI.
