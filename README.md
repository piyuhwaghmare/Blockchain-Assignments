## 🚀 About This Repository

Welcome to my comprehensive portfolio of Web3 engineering assignments. This repository serves as a chronological journey through the foundational and advanced concepts of blockchain technology. 

From deploying immutable logic on Ethereum to scaling with Layer 2 solutions, engineering decentralized frontends, storing unstoppable data, and building autonomous governance models—this collection demonstrates a full-stack approach to the decentralized web.

---

## 📂 Repository Architecture

This project is divided into five core modules, each tackling a critical pillar of modern blockchain development:

### ⚡ [Assignment 1: Foundations of Trustless Systems](./Assignment1)
**Smart Contract Deployment**
The genesis of this journey. This module focuses on writing, compiling, and deploying foundational Solidity smart contracts to the Ethereum blockchain (Sepolia Testnet), establishing the basics of state manipulation, gas optimization, and immutable logic.

### 🌌 [Assignment 2: Scaling for the Future](./Assignment2)
**Polygon Network Integration**
Addressing the blockchain trilemma by taking our contracts to Layer 2. This assignment demonstrates how to bypass high Layer 1 gas fees and network congestion by deploying to the Polygon (MATIC) network for high-speed, scalable transactions.

### 🌉 [Assignment 3: Bridging the Gap](./Assignment3)
**Web3 UI / DApp Integration**
Smart contracts are only as good as their user experience. This module features a custom frontend application that seamlessly connects human intent to blockchain execution, allowing users to sign and broadcast transactions directly via MetaMask.

### 📦 [Assignment 4: Unstoppable Data](./Assignment4)
**Decentralized File Storage (IPFS)**
Moving beyond the limitations of on-chain storage. This assignment implements a robust Node.js backend to upload, pin, and retrieve files securely using the InterPlanetary File System (IPFS) and the Pinata network, ensuring data remains tamper-proof and decentralized.

### 🏛️ [Assignment 5: The Power of Consensus](./Assignment5)
**Decentralized Autonomous Organization (DAO) Framework**
Redefining digital governance. The final module explores the creation of a DAO smart contract, empowering a community to propose, vote on, and automatically execute decisions in a completely trustless and transparent environment.

## 🛠️ Technology Stack

This repository leverages a modern Web3 development stack:

* **Smart Contract Language:** Solidity
* **Blockchain Networks:** Ethereum (Sepolia), Polygon (Amoy/Mumbai)
* **Development & Testing:** Remix IDE, Hardhat / Truffle
* **Wallet & Authentication:** MetaMask
* **Frontend Integration:** Ethers.js / Web3.js, HTML/CSS/JavaScript
* **Decentralized Storage:** IPFS, Pinata API, Node.js
